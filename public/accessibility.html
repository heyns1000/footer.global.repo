Accessibility Features

Our website includes features designed to improve accessibility for all users:

  • **Keyboard Navigation:** All interactive elements can be navigated using a keyboard.
  • **Screen Reader Compatibility:** Our content is structured to be compatible with popular screen readers.
  • **Color Contrast:** We strive to maintain sufficient color contrast for readability.
  • **Resizable Text:** Users can adjust text size through browser settings.
  • **Descriptive Alt Text:** Images and other non-text content include descriptive alternative text.
  • **Consistent Layout:** A predictable and consistent layout helps users navigate the site easily.
